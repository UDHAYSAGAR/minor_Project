# Retail Sales Performance Analysis Dashboard

A modern and beginner-friendly Retail Sales Performance Analysis Dashboard developed as a B.Tech Minor Project.

The application allows users to upload retail sales datasets and instantly analyze sales performance through interactive charts, KPI cards, and automated business insights.

## Features

- Upload CSV sales data
- Total Sales, Profit, Orders, and Average Sales KPIs
- Monthly Sales Trend Analysis
- Brand-wise Sales Analysis
- Region-wise Performance Analysis
- Mobile Model Sales Comparison
- Profit Analysis Dashboard
- Automated Business Insights
- Responsive and Professional UI
- Real-time Data Visualization using Chart.js

## Technologies Used

- Python
- Flask
- Pandas
- HTML5
- CSS3
- JavaScript
- Chart.js

## Dataset

The project uses a realistic mobile sales dataset containing:

- Offline Stores (LOT Mobiles, Big C, Sangeetha, Poorvika, Reliance Digital, Croma)
- Online Platforms (Amazon, Flipkart)
- Andhra Pradesh Cities
- Popular Mobile Brands
- Festival Season Sales Trends (Dussehra & Diwali Peaks)

## Project Objective

To help businesses understand sales trends, identify top-performing products, evaluate regional performance, and make data-driven decisions through easy-to-understand visual analytics.
